# Delta Lake (delta-lake)
Delta Lake is an incubation-stage project under LF AI and Data Foundation providing an open source storage framework for building Lakehouse architectures. Originally contributed by Databricks, it adds reliability, quality, and performance to data lakes with ACID transactions, schema enforcement, and time travel.
